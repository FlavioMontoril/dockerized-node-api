# Node + Prisma + MySQL Docker Demo

Este projeto é um exemplo de **containerização de uma API Node.js** com **Prisma** e **MySQL** usando **Docker multi-stage**.  

O objetivo é demonstrar:

- Criação de uma API Node.js usando TypeScript.
- Conexão com MySQL usando Prisma.
- Configuração de **Docker multi-stage build** para produção.
- Uso de **docker-compose** para subir API + banco.
- Geração automática do Prisma Client durante o build.

## Tecnologias

- Node.js + TypeScript
- Prisma ORM
- MySQL
- Docker / Dockerfile / Docker Compose
